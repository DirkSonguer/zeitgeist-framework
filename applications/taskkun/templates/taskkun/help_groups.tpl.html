Gruppen anlegen und verwalten

Gruppen sind die Grundlage für die Projektverwaltung in Taskkun. Die Projektaufgaben werden über die Aufgabenabläufe an Gruppen verteilt, deren Gruppenteilnehmer sie jeweils bearbeiten.

Als Administrator können Sie die Gruppen verwalten, die Ihrem Projektteam angehören. Sie können neue Gruppen anlegen, bestehende Gruppen und deren Attribute verwalten oder Gruppen aus dem System entfernen.

Mehr Informationen über Administratoren erhalten Sie in dem Kapitel "Benutzerrollen".

Gruppenübersicht

In der Gruppenübersicht erhalten Sie eine Liste mit allen aktuell im System zur Verfügung stehenden Gruppen. Zu jeder Gruppe gibt es eine Beschreibung und eine Übersicht, wie viele Benutzer der jeweiligen Gruppe angehören und wie viele aktive Aufgaben zur Zeit von dieser Gruppe bearbeitet werden.

Neue Gruppe anlegen

Über der Gruppenliste befindet sich der Punkt: "Neue Gruppe hinzufügen". Mit diesem können Sie eine neue Gruppe in Ihrem System erstellen.

Grundsätzlich besteht eine Gruppe nur aus dem Namen und ihrer Beschreibung. Die Gruppe ist direkt nach der Erstellung im System verwendbar: sie können die Gruppe in Aufgabenabläufen einbinden und Benutzer der Gruppe zuweisen.

Bestehende Gruppen bearbeiten

Bereits bestehende Gruppen können in der Übersicht mit dem Button "Gruppe bearbeiten" geändert werden. Die Änderungen wirken sich direkt auf alle Aufgabenabläufe und Mitglieder der Gruppe aus.

Gruppen löschen

In der Gruppenübersicht können einzelne Gruppen mit dem Button "Gruppe löschen" aus dem System entfernt werden. Bitte beachten Sie, dass nur Gruppen entfernt werden können, die kein Teil eines Aufgabenablaufs und keine aktiven Aufgaben mehr besitzen. Beenden Sie zunächst alle aktiven Aufgaen und entfernen Sie anschließend die Gruppe aus allen Aufgabenabläufen.
