Setup application

Here you can set up and configure your Zeitgeist project.

You will see the currently active project in the headline. You can configure and set the currently active project in the ZGA configuration.

On this global scale you can define what kind of userdata and what userroles are available to your application. This should be designed during development as your application has to interface the userdata or check for userroles in order to work. Refactoring the data structures on a live set of users and their data might have unforseen consequences in your application.
