Link a new project

To create a link to a Zeitgeist project, this tool needs to connect to its database.

Please enter the credentials of your database carefully to avoid configuring the wrong application. It might be a good idea to create seperate users for every project. On submit the database will be checked for compatibility.

Instead of linking an existing Zeitgeist project, you can create a completely new one by checking "Create Zeitgeist database". If you do, the tool will create a new Zeitgeist database layout inside the given database.

Use the buttons to either submit your changes or get back to the project overview without saving.
