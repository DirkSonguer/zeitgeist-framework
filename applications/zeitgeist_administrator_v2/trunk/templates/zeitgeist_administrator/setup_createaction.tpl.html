Editing actions

The name of the action acts as the identifier for the controller. It tries to call a method with the same name as the action inside the module class. You should not choose weird action names - remember you have to name your methods accordingly.

The description of a action just acts as a reminder for yourself and is not used by the framework.

Inactive actions are ignored by the controller. If a user tries to use an inactive action, an error is thrown.
