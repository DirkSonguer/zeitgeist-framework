
Welcome to the Zeitgeist Framework examples.

If you have questions about the examples or the source feel free to ignore this email and figure it out yourself: dirk@zeitalter3.de

If you don't, ask nicely.


_Prerequisites_
---------------

* Some webserver
* PHP 5.2 or newer
* A MySQL 5 server
* A copy of the Zeitgeist Framework (see http://zeitgeist-framework.googlecode.com/)


_Setup_
-------

* Create a new folder on you webserver, for example "yourwebroot/zeitgeist_examples".
* Copy the contents of this directory into the new folder
* Start your browser, call the /install/ subdirectory and follow the instructions (http://127.0.0.1/zeitgeist_examples/install/)
* Call the index.php in the root directory. Should be something like this: http://127.0.0.1/zeitgeist_examples/index.php
* Click around
* Read the source
