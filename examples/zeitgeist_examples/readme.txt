Welcome to the Zeitgeist Framework examples.

If you have questions about the examples or the source feel free to ignore this email and figure it out yourself: dirk@songuer.de

If you don't, ask nicely.


_Prerequisites_
---------------

* Some webserver
* PHP 5.2 or newer
* A MySQL 5 server


_Setup_
-------

* Copy this directory to your htdocs directory (or in some folder there)
* Dump ./_additional_files/zeitgeist_examples.sql in a new table with the same name on your webserver
* Configure the database access in ./configuration/application.configuration.php
* Start your browser and call the index.php. Should be something like this: http://127.0.0.1/zeitgeist_examples/index.php
* Click around
* Read the source
