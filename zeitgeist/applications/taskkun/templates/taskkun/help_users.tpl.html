Benutzer anlegen und verwalten

Als Administrator können Sie die Benutzer verwalten, die Ihrem Projektteam angehören. Sie können neue Benutzer anlegen, bestehende Benutzer und deren Attribute verwalten oder Benutzer deaktivieren bzw. löschen.

Mehr Informationen über Administratoren erhalten Sie in dem Kapitel "Benutzerrollen".

Benutzerübersicht

In der Benutzerübersicht erhalten Sie eine Liste mit allen angelegten Benutzern Ihres Projektteams. Zu jedem Benutzer wird seine Benutzerolle und die Anzahl seiner aktiven, übernommenen Aufgaben angezeigt.

Die Übersichtsliste kann nach Gruppenzugehörigkeit und Benutzerrolle gefiltert werden. Bitte beachten Sie, dass jeweils nur ein Filter gleichzeitig aktiv sein kann.

Neue Benutzer anlegen

Über der Benuzerliste befindet sich der Punkt: "Neuen Benutzer hinzufügen". Mit diesem können Sie einen neuen Benutzer zu Ihrem Projektteam hinzufügen.

Die Informationen zu dem neuen Benutzer sind in 3 Blöcke geteilt:

Benutzerdaten

Hier können allgemeine Informationen zu dem Benutzer eingetragen werden. Die angegebene E-Mail-Adresse des Benutzers wird zur Anmeldund des Benutzers genutzt. Bitte beachten Sie, dass jede E-Mail nur einmal im System verwendet werden kann. Dies gilt übergreifend für alle Projektteams.

Vorname und Name des Benutzers werden jeweils bei Informationen bezüglich Aufgaben und bei den Reports angezeigt.

Weitere Angaben, wie Straße, Hausnummer etc. sind freiwillig und sind zum Beispiel sinnvoll bei verteilten Projektteams.

Benutzereigenschaften

Benutzerrolle spezifiziert die Rolle, die der neue Benutzer im System einnehmen wird. Mehr Informationen bezüglich Nutzerrollen erhalten Sie in dem Kapitel "Benutzerrollen".

Gruppen spezifiziert die Projektgruppen, in denen der neue Benutzer eingebunden wird. Mehr Informationen zu Gruppen finden Sie in dem Kapitel "Gruppen".

Passwort

Das initiale Passwort für den neu angelegten Benutzer. Der Benutzer kann dies anschließend selbst in seinen Einstellungen ändern.

Bestehende Benutzer bearbeiten

In der Benutzerübersicht befindet sich rechts zu jedem Benutzer das Icon "Benutzer bearbeiten". Hier können sie jeden Aspekt des Benutzers bearbeiten. Bitte beachten Sie, dass Änderungen an der Zugehörigkeit des Benutzers zu Benutzerrollen oder Benutzergruppen Probleme verursachen können, wenn der Benutzer gerade aktiv im System arbeitet. Es ist empfehlenswert, dass sich der betroffene Benutzer aus- und wieder einloggt.

Benutzer löschen

In der Benutzerübersicht ist für jeden Benutzer ganz rechts ein Icon "Benutzer löschen" vorhanden. Damit können Sie Benutzer und deren Beiträge vollständig aus dem System entfernen.

Bitte beachten Sie, dass mit dem Löschen eines Benutzers auch alle Aufgabenbeschreibungen und Tätigkeiten des Benutzers gelöscht werden. Der Benutzer wird nicht mehr in Reports geführt. Abgeschlossene Aufgaben werden also in so fern verfälscht, als dass die Tätigkeiten des Benutzers gelöscht werden.

Es werden alle "Spuren" des Benutzers aus dem System entfernt. Falls Sie die Aufgabenbeschreibungen und Tätigkeiten zur Dokumentation aufbewahren wollen, deaktivieren Sie den Benutzer statt dessen.

Benutzer deaktivieren

Links neben dem Benutzernamen zeigt eine Ampel den Status des Benutzer an: grün für aktiv, rot für deaktiviert. Mit einem Klick auf die Statusanzeige können Sie den Status des Benutzers ändern.

Ein deaktivierter Benutzer kann sich nicht mehr in Taskkun einloggen. Seine Aufgaben, Aufgabenbeschreibungen, und festgehaltene Tätigkeiten bleiben jedoch erhalten.
