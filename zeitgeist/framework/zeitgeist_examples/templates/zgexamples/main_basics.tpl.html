Zeitgeist Examples - Overview

Welcome to Zeitgeist, a PHP based multi purpose framework for web applications.

Following are examples to get you started understanding and using Zeitgeist. This collection serves as tutorial as well as reference manual and style guide.

Maybe you already looked at the code you just installed - that's ok. When looking at the sources of the examples, please keep the following things in mind:

  1. This is not a real application but merely a collection of examples and tutorials.
  2. Many functionalities are used in a dumbed down way. These are examples after all. However we try to present best practises for each framework module.
