About the Zeitgeist Administrator

This is the main page of the Zeitgeist Administrator v2. This tool will help you create and manage projects based on the Zeitgeist framework.

Note that this is a pretty simple tool with very little failsafes. So be careful when editing or it will bring down your application!

Note that you have to link and configure your Zeitgeist projects before you can use them.
