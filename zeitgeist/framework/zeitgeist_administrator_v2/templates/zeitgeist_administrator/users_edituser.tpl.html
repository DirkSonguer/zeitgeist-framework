Quickhelp

Here you can see all the users of your application.

By default the list is sorted by ID, which should in most cases also represent the order of creation in descending order. Use the row titles to sort the list to your liking.

Use the icons to the right to either edit a user or delete it. Notice that you can't restore deleted users with this tool.

The search will filter by ID as well as the username as you type. Note that it's case insensitive. Use the "Clear" button to reset the filter and show all users.
