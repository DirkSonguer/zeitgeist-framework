Welcome to the Zeitgeist administration tool. The purpose of this tool is to help you developing applications with the Zeitgeist Framework, as well as act as a central control and monitoring platform for your running Zeitgeist applications.

Setup application

This section helps you setting up your application, for example adding actions and modules to the database etc.

Application users

This section lets you manage the users of your application as well as manage their rights and roles.

Configure ZGA

Here you can manage the Zeitgeist administrator tool.
